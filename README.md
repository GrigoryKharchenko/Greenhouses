# Greenhouses

## Описание
Тестовое техническое задание. На текущий момент сделаны экраны авторизации, регистрации, профиля и редактировние профиля.

## Стэк технологий
- MVVM
- Retrofit, okhhtp
- Dagger
- Coroutines
- ViewBindin
- DataStore
